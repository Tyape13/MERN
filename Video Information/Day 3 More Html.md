### 1 - Inline vs Block Element
Block elements Starts - OCCUPY NEW LINE ALWAYS. Starts with First Character of Line. They occupy whole Block.
Inline Elements - DON'T OCCUPY NEW LINE. Just stay in the line itself.
    eg - 3 cycles in 3 rows, 3 cycles starts one after another in same row.
    h1 tags, p tags are block elements. 
    span tags, anchor tags, img tags are inline elements.
